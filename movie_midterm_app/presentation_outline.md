# CineBox Presentation Outline

## Slide 1 - App Name
CineBox: Browse famous movies in a clean Flutter app.

## Slide 2 - Folders & Files
- lib/screens: splash, onboarding, login, sign-up, home, settings, details
- lib/widgets: reusable UI widgets like AppTextField, PrimaryButton, MovieCard
- lib/models: Movie model
- lib/data: hardcoded real movie data
- assets/translations: English and Arabic JSON files
- assets/images: SVG logo, onboarding images, and local movie poster-style images

## Slide 3 - Packages Used
- easy_localization: English / Arabic language support
- flutter_svg: SVG images and poster assets
- flutter_animate: simple animations
- introduction_screen: onboarding pages
- curved_navigation_bar: bottom navigation
